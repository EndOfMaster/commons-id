# ID construction tool
1. uuid
2. mongoId
3. clusterId

## Maven Repository
    https://maven.pkg.github.com/EndOfMaster/maven-repo
### Maven Import
```xml
<dependency>
  <groupId>com.endofmaster</groupId>
  <artifactId>commons-id</artifactId>
  <version>1.0.2</version>
  <type>pom</type>
</dependency>
```
### Gradle Import
```groovy
compile 'com.endofmaster:commons-id:1.0.2'
```
